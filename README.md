# EgyTravel Tourism System

## 📌 Overview

EgyTravel is a tourism itinerary planning system designed to generate realistic travel plans using AI and structured tourism data.

This project focuses on the **database design and system structure** supporting itinerary generation.

---

## 🗄️ Database Design

* Designed relational database schema for tourism data
* Managed entities such as users, locations, attractions, and itineraries
* Ensured data consistency and scalability

---

## 🧠 System Features

* Tourism itinerary generation
* Data-driven recommendations
* Structured storage for locations and trips

---

## 🛠️ Tech Stack

* SQL / MongoDB
* Database Design
* ERD Modeling

---

## 📊 Project Highlights

* Built database from scratch
* Designed ERD and schema
* Supports AI-based itinerary systems
